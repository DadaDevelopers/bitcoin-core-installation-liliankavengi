# Bitcoin Core Installation

## Environment
- OS: Kali Linux 2025.4
- Bitcoin Core Version: v27.0.0

## Steps
1. Downloaded Bitcoin Core v27.0.0
2. Extracted and installed binaries to /usr/local/bin
3. Configured bitcoin.conf with regtest mode
4. Started bitcoind in regtest mode
5. Created wallet: lilianwallet
6. Generated address and mined 101 blocks
7. Sent 10 BTC transaction and confirmed it

## Result
Successfully installed and ran Bitcoin Core in regtest mode.
